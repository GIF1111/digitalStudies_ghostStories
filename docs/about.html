About This Project

Exploring emotion and evolution in Victorian ghost stories

Research Questions

1. How do Victorian ghost stories use language to evoke emotions such as fear, suspense, and relief, and how do those emotional patterns shift throughout a story?

2. Over time, do overall tones change in Victorian ghost stories?

How We Built the Website

  1. To build the Digital Ghost Stories website, we first selected a collection of short stories as the foundational texts. These stories were then converted into XML, allowing us to extract key elements like characters, tones, and story content. Each story was tagged with XML markup to categorize and identify relevant sections such as dialogue, paragraphs, and notes, making the content easier to structure and display on the site.
  2. We used XSLT to transform the XML data into HTML story pages. XQuery was used to generate visualizations — including tone graphs, character analysis charts, and story timelines — all automatically driven by the data in the XML files.
  3. For styling, we applied CSS to create a clean, gothic-inspired design that complements the ghostly theme. The site uses a structured layout with a navigation bar for easy access to major sections and a main content area to highlight each story and analysis page.
